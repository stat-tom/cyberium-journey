# Lesson 5 - Operations on Files and Directories

## Status
- Completed

## Overview
- Notes and examples for working with files and directories.

## Notes
- List of commands covered in this lesson:

	- `touch` - create empty files
	- `mkdir` - create directories
	- `rm` - remove files and directories
	- `tree` - display the directory structure
	- `cp` - copy files and directories
	- `mv` - move files and directories
	- `echo` - display text
	- `cat` - display file contents
	- `find` - search for files
	- `wc -l` - count lines
	- `sort` - sort content

- Operators covered in this lesson:

	- `>` - output redirection (overwrite)
	- `>>` - output redirection (append)
	- `|` - pipe (pass output)
	- `<` - input redirection
	- `2>` - error redirection
