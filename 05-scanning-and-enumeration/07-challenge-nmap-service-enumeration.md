# Lesson 5 Challenge - Nmap Service Enumeration

## Status
- Planned

## Objective
- Practice using service detection and basic enumeration to understand what is running on a target host.

## Tasks
- Task 1:
	- Identify the most relevant open ports from a previous scan.
	- Run version detection and record the most likely service information.
- Task 2:
	- Review banners, certificates, headers, or script output where available.
	- Note which details are high confidence and which are only probable.
- Task 3:
	- Choose one exposed service and describe what follow-up testing would make sense next.
	- Explain how the enumeration result changes the target priority.

## Reflection Questions
- Which service details were easy to confirm and which were ambiguous?
- Did enumeration reveal anything unexpected compared with the initial port scan?
- What would you report first to a client from these results?

## Notes
- The goal is to convert technical output into reliable conclusions and useful next actions.
