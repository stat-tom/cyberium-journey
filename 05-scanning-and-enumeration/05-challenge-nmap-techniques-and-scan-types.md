# Lesson 4 Challenge - Nmap Techniques and Scan Types

## Status
- Planned

## Objective
- Practice selecting appropriate Nmap scan types and interpreting the results in a legal lab environment.

## Tasks
- Task 1:
	- Discover whether the target host appears alive.
	- Compare at least two discovery approaches and note any difference in results.
- Task 2:
	- Run a TCP scan against the target and record which ports appear open, closed, or filtered.
	- Consider how firewall behavior may affect the outcome.
- Task 3:
	- If permitted in the lab, compare a connect scan and a SYN scan.
	- Note which method is noisier, faster, or reveals different details.

## Reflection Questions
- Which scan type gave the clearest result in this environment?
- Which findings still need manual validation?
- How might the results change if the target were behind stricter filtering?

## Notes
- The challenge is about choosing and interpreting techniques, not just running commands mechanically.
